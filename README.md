# Darts 501 Game (Java)

A console-based Darts 501 game written in Java. Game follows official Darts 501 rules, including the bust rule and three-dart turns.
This game allows you to play against your friends, track scores, turns, and player statistics like three-dart averages and highest turn scores.

## Main Features
- Two-player game with customizable names
- Decide who throws first via **bull up** or choice
- Score tracking with **bust rule** to ensure it follows darting rules
- Player statistics:
  - Turns taken
  - Darts thrown
  - Highest turn
  - Three-dart average
- End-of-game summary with the winners stats

## How to Run
1. Clone or download the repository.
2. Open in VS Code (or any Java IDE).
3. Run `Main.java`.
4. Follow the on-screen prompts to play.

## Gameplay Instructions
1. Enter your name when prompted.
2. Choose who throws first via bull up (closest to the bullseye goes first) or by selecting a player.
3. Each player throws three darts per turn.
4. Bust rule: if you score more than your remaining points, your turn is canceled, and your score reverts.
5. Continue until a player reaches exactly zero to win (Must finish the game on a double).
